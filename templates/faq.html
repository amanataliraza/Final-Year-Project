Frequently Asked Questions

Where can I find all the results after running the program?
For all the graphs, they will be subpages in the visualization page.
What does the bar plot mean?
The bar plot shows the top 10 largest genes that were present in this study.
How long does it take to run the prediction?
The prediction usually takes a few seconds to complete.
Can I use my own dataset for prediction?
Currently, the program only supports the predefined dataset for fibromyalgia prediction.
Are the results confidential?
Yes, all the results and data are kept confidential and are not shared with any third parties.
Is the program available for offline use?
No, the program requires an internet connection to access the necessary resources for prediction.
Can I download the generated graphs and results?
Yes, you can download the generated graphs and results as image files or in a tabular format.
