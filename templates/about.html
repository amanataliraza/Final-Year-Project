About Us

This project aims to predict whether a person has fibromyalgia or not by utilizing gene expression patterns. The gene expression data is retrieved from the Gene Expression Omnibus (GEO), a public repository that houses a vast collection of gene expression datasets.

The project was implemented using Python and leveraged various tools and libraries, including Jupyter Notebook, Anaconda Navigator, Pandas, NumPy, scikit-learn, TensorFlow, Keras, and Matplotlib. Jupyter Notebook provided an interactive computational environment for data exploration and analysis. Anaconda Navigator facilitated the management of project dependencies and environments.

Pandas and NumPy were utilized for data manipulation and preprocessing tasks. The scikit-learn library offered a wide range of machine learning algorithms and utilities for model development, evaluation, and performance metrics. TensorFlow and Keras provided a powerful framework for building and training neural network models.

Throughout the project, data visualization was carried out using Matplotlib, enabling the generation of informative plots and charts to gain insights from the data. By analyzing the expression levels of specific genes, the machine learning model developed in this project aims to provide valuable insights and potential diagnostic capabilities for identifying fibromyalgia.
